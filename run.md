# Running to-do-list cli

1. Clone the repo
```
git clone https://github.com/3nya/todo-list-cli.git
```

2. Install
```
cd todo-list-cli/
cargo install
```

3. Using todo-list-cli
For list of commands
```
cargo run help
```
All commands should follow this format: `cargo run <command>`

To view your todo-list
```
cat todo.txt
```
